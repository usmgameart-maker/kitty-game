# Математическое приключение с Китти

Образовательная игра по математике для детей на Phaser 3.

Игрок вместе с котиком Китти проходит уровни по сложению, вычитанию, умножению и делению, собирает артефакты и выполняет миссию спасения.

## Играть онлайн

https://usmgameart-maker.github.io/kitty-game/

## Технологии

- [Phaser 3](https://phaser.io/)
- JavaScript (ES6 modules)
- HTML5 / CSS3
